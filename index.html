Tiny Docs
All your Docs in a single file

Tiny Docs is a simple and lightweight documentation template that allows you to write documentation in markdown and have it rendered as a static site.

Why Tiny Docs?

Single File

Just one HTML file! No need for a server, complex setup, or build step.

Markdown Support

Write your documentation in Markdown and have it rendered beautifully as a static site.

Zero Dependencies

No need for a build step or package manager. Tiny Docs works out of the box.

Discover Tiny Docs Builder

It's a simple yet powerful tool that allows you to write and customize your documentation in a visual way. It is currently in beta, but you can try it out now.

Open Source

Tiny Docs is open source and available on GitHub. Feel free to contribute, suggest features, or report bugs.
